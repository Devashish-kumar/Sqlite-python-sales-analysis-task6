##  SQL Queries Implemented

### ** Query 1: Product Sales Summary**
```sql
SELECT 
    product,
    SUM(quantity) AS total_qty,
    SUM(quantity * price) AS revenue,
    ROUND(AVG(price), 2) AS avg_price
FROM sales 
GROUP BY product 
ORDER BY revenue DESC;
```

**Purpose**: Calculate total quantity sold and revenue for each product

### ** Query 2: Category Performance Analysis**
```sql
SELECT 
    category,
    COUNT(*) AS num_transactions,
    COUNT(DISTINCT product) AS num_products,
    SUM(quantity) AS total_qty,
    SUM(quantity * price) AS revenue,
    ROUND(AVG(quantity * price), 2) AS avg_transaction_value
FROM sales 
GROUP BY category 
ORDER BY revenue DESC;
```

**Purpose**: Analyze performance across different product categories

### ** Query 3: Sales Representative Performance**
```sql
SELECT 
    sales_rep,
    COUNT(*) AS transactions,
    SUM(quantity) AS total_items_sold,
    SUM(quantity * price) AS total_revenue,
    ROUND(AVG(quantity * price), 2) AS avg_sale_value
FROM sales 
GROUP BY sales_rep 
ORDER BY total_revenue DESC;
```

**Purpose**: Track individual sales rep performance and productivity

### ** Query 4: Daily Sales Trend**
```sql
SELECT 
    sale_date,
    COUNT(*) AS transactions,
    SUM(quantity) AS items_sold,
    SUM(quantity * price) AS daily_revenue
FROM sales 
GROUP BY sale_date 
ORDER BY sale_date;
```

**Purpose**: Identify sales trends and patterns over time

### ** Query 5: Overall Business Summary**
```sql
SELECT 
    COUNT(*) AS total_transactions,
    COUNT(DISTINCT product) AS unique_products,
    COUNT(DISTINCT customer_id) AS unique_customers,
    SUM(quantity) AS total_items_sold,
    SUM(quantity * price) AS total_revenue,
    ROUND(AVG(quantity * price), 2) AS avg_transaction_value,
    MIN(sale_date) AS first_sale_date,
    MAX(sale_date) AS last_sale_date
FROM sales;
```

**Purpose**: Generate comprehensive business overview and KPIs

```python
import sqlite3
import pandas as pd
import matplotlib.pyplot as plt

conn = sqlite3.connect('sales_data.db')

query = "SELECT product, SUM(quantity) AS total_qty, SUM(quantity * price) AS revenue FROM sales GROUP BY product"
```

### **1. How did you connect Python to a database?**


**Benefits**:

- File-based database (portable)

### **2. What SQL query did you run?**

**Main Query**:
```sql
SELECT product, SUM(quantity) AS total_qty, SUM(quantity * price) AS revenue 
FROM sales GROUP BY product ORDER BY revenue DESC
```

**Components**:
- `SELECT`: Chooses columns to return
- `SUM()`: Aggregates quantity and calculates revenue
- `GROUP BY`: Groups rows by product
- `ORDER BY`: Sorts results by revenue (descending)

### **3. What does GROUP BY do?**

**Function**: Aggregates rows with same values into summary rows

**Example**:
```
Without GROUP BY:
Laptop | 2 | 999.99
Laptop | 3 | 999.99

With GROUP BY product:
Laptop | 5 | 4999.95 (total qty | total revenue)
```

### **4. How did you calculate revenue?**

**Formula**: `SUM(quantity * price) AS revenue`

**Process**:
1. Multiply quantity × price for each row
2. SUM() aggregates all individual calculations
3. GROUP BY ensures summation per product/category
